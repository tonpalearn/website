Pipeline Setup · 8 Steps

ขั้นตอนการ setup AI Web App Pipeline ครบจบใน 1 หน้า — Claude Cowork → Claude Code → Github → Vercel → Supabase

TON PA LEARN
tonpalearn.com/pipeline
PHASE A · PLAN
1
Claude Cowork Project
เปิด claude.ai → Projects → + Create
ตั้งชื่อ project, อัปโหลด reference files
➜ claude.ai
2
Project Instructions
ใน Project → Custom instructions
เขียน system prompt: บทบาท · style · ภาษา · context
➜ Project settings
3
CLAUDE.md
สร้างไฟล์ CLAUDE.md ใน root folder
ใส่: stack · convention · do/don't · paths
➜ ./CLAUDE.md
PHASE B · CODE
4
HERO STEP
Claude Code Terminal
เปิด terminal → cd ไปที่ project folder
พิมพ์ claude → เริ่ม session
Claude อ่าน CLAUDE.md auto
สั่งงานเป็นภาษาไทยได้เลย
$ ตัวอย่างคำสั่ง
"สร้าง landing page Hero +
Course list + CTA"
PHASE C · INFRASTRUCTURE
5
Github Repository
เข้า github.com → New repo
ใช้ Claude Code git push upload code
➜ github.com/USERNAME/REPO
6
Vercel Deployment
vercel.com → Import Github repo
Auto-deploy → ได้ project.vercel.app
(option) Settings → Domain → ซื้อโดเมนเอง
➜ vercel.com
7
Supabase Database
supabase.com → New project
เก็บ 3 keys: URL · anon · service_role
ใส่ใน Vercel → Settings → Env Vars
➜ supabase.com → Settings → API
8

Daily Workflow — Auto-Deploy
